ztree系列组件【提供于标准版及以上】

zTree 一个依靠 jQuery 实现的多功能 “树插件”。优异的性能、灵活的配置、多种功能的组合是 zTree 最大优点，在使用layui的不少用户对于tree组件的缺失需要一些替代，在LarryMS中相继会推出trees组件(即larryTree)，也可以使用layui官方扩展平台中用户分享的组件，亦或等待layui的tree模块，但在此提供较为经典的ztree也是一种选择方案(ztree系列组件：ztree、ztreeCheck、ztreeExedit、ztreeExhide)，下面是通过对ztree系列模块进行二次封装后实现的示例展示：（它的用法可在larryms专属vip文档中查看用法示例，也可以直接查看ztree的官方文档，只是在larryms中让其符合了layui的模块加载规范，按需加载使用）

    ztree在LarryMS框架模板中的使用

    ztree功能强大，可以实现诸如：下拉框选择，多选、节点隐藏，一次性加载大数据量，分批异步加载，自定义风格，用于右键菜单，对节点进行增删改移动复制等等，可在larryms专属vip文档中查看，下面是larryms使用ztree的代码段举例： 

    layui.use('ztree', function() 
    var $ = layui.$,
        ztree = layui.ztree;
    var setting_a = {data: {simpleData: {enable: true}}},
        zNodes_a = [{...},{...}];//限于篇幅这里列出的是数据定义格式，具体查看文档介绍
    ztree.init($("#treeDemo"), setting_a, zNodes_a);
    //#其他相关代码
});
