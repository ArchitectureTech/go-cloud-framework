在LarryMS框架模板中forms组件新增有表单左右穿梭框功能：

1、支持自定义数据源，默认json格式:直接赋值或url
2、穿梭框内容外观均可完全自定义，包括主题设置、左侧标题、是否开启智能输入补全、右侧标题，支持高度设定等，无需额外组件引入，只依赖于forms模块
3、支持选项拖动、双击、单击进行选择、支持全选、支持标签飞入飞出动画、支持多层级树形选择
4、支持搜索补全选择操作、选项例外项可自定义

备注：LarryMS框架在日常开发中遇到需要左右列表的分配，分配菜单、分配权限等场景时会用到穿梭选择功能，如下演示：
transfer功能演示：拖动、双击、单击、标签动画

javascript代码如下：

//===============JS代码：==============
layui.use('forms',function(){
    var forms = layui.forms;
    //若设置data-source属性，支持自动渲染，下方是手动渲染代码
    forms.transfer({
        elem:'#test',
        source:'../../larryms/data/demo/transfer.json'
    }); 
    forms.on('submit(baseTransfer)',function(data){
        larryms.msg(JSON.stringify(data.field));
        return false;
    });
});
