安装方法    
----
1、go get https://gitee.com/yellbuy/YbGoCloundFramework    
2、创建mysql数据库，并将YbGoCloundFramework.sql导入    
3、修改config 配置数据库    
4、运行 go build    
5、运行 ./run.sh start|stop


前台访问：http://your_host:8081
